## 🎮 Game Rules
- Computer chooses one secret number between 1 and 50  
- User gets total 10 attempts to guess the correct number  
- User enters a guess between 1 and 50  
- User can enter `0` anytime to exit the game  
- If the guessed number is correct → user wins and game stops  
- Correct guess also counts as one attempt  
- If the guessed number is already guessed before → warning shows and not count in attempts 
- If the guessed number is very near to the secret number (within ±2) → hint shows `Very Close!`  
- If the guessed number is greater than the secret number → hint shows `Too High`  
- If the guessed number is smaller than the secret number → hint shows `Too Low`  
- Remaining attempts are shown after every valid guess  
- If all attempts finish and the number is still not guessed → `Game Over!` message appears  
- Final result depends on win, exit, or attempts finished
